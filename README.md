# Cisco AI & Machine Learning — Student Repo (June 2026)

Six-day hands-on course: **logs, tickets, Gen AI, RAG, MCP, and agentic workflows**.

## Setup (once)

**Windows:** `setup_student_env.bat` then `start_jupyter.bat`  
**Mac / Linux:** `./setup_student_env.sh` then `./start_jupyter.sh`

Kernel: **Python (cisco-aiml-lab)**

## Six days

| Day | Folder | Focus |
|-----|--------|--------|
| 1 | `hands-on/01-data-science-introduction/` | AI / Gen AI landscape |
| 2 | `hands-on/02-python-for-data-science/` | Logs, tickets, trends |
| 3 | `hands-on/03-classification-interpretation/` | Ticket analysis, prompts |
| 4 | `hands-on/04-distance-mlops/` | Hugging Face, local SLMs |
| 5 | `hands-on/05-unsupervised-learning/` | RAG, MCP, agents |
| 6 | `hands-on/06-anomaly-detection/` | Security, capstone |

**Data:** `data/support-ops/`

Docs: `docs/student-quick-start.md` · `docs/student-troubleshooting.md`
