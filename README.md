# AI & Machine Learning — Cisco Training (June 2026)

Python 3.10+. Your instructor will walk you through each day — use this repo for setup and lab notebooks.

## Setup (once)

**Windows:** `setup_student_env.bat` then `start_jupyter.bat`  
**Mac / Linux:** `./setup_student_env.sh` then `./start_jupyter.sh`

Jupyter kernel: **Python (cisco-aiml-lab)**

Clone:

```bash
git clone https://github.com/atingupta2006/10-June-26-AI-ML-Cisco.git
cd 10-June-26-AI-ML-Cisco
```

More detail: `docs/student-quick-start.md` · Problems: `docs/student-troubleshooting.md`
