# Student quick start

## 1. Clone

```bash
git clone https://github.com/atingupta2006/10-June-26-AI-ML-Cisco.git
cd 10-June-26-AI-ML-Cisco
```

## 2. Setup (once)

| OS | Command |
|----|---------|
| Windows | `setup_student_env.bat` |
| Mac / Linux | `chmod +x setup_student_env.sh && ./setup_student_env.sh` |

Needs Python 3.10+.

## 3. Jupyter

| OS | Command |
|----|---------|
| Windows | `start_jupyter.bat` |
| Mac / Linux | `./start_jupyter.sh` |

Pick kernel **Python (cisco-aiml-lab)**.

## 4. Labs

Your instructor will say which day to open. Path pattern:

`hands-on/<day-folder>/notebooks/lab01_….ipynb` … `lab06_….ipynb`

Use the CSV files in `data/support-ops/` for all labs.

Problems? See `student-troubleshooting.md` in this folder.
